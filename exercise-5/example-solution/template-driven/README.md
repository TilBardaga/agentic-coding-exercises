# Example solution (reference)

This folder contains **example** implementations of a `/plan-feature`-style command. They illustrate one possible depth and structure; your command may be shorter, longer, or organized differently.

| File | Description |
|------|-------------|
| **`plan-feature.md`** | Full slash-command body: multi-phase research, large embedded template, output under `.agents/plans/`. Uses **`AGENTS.md`** for project rules (workshop convention). |
| **`alternative_solution.md`** | Same core template with an explicit **phase-by-phase** breakdown (how each phase fills parts of the template). |

Use these as patterns to steal from, not as something to copy blindly.
