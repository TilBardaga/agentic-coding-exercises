# Exercise 2 — PIV on the catalog app

You apply the **PIV loop** (Plan → Implement → Validate) in **`exercise-2/app/`**, on the **same** filtering feature as **[Exercise 1](../exercise-1/README.md)**—with **ticket-style specs** in the assignment below.

## Quick start

From **`exercise-2/`**:

```bash
cd app/backend
uv venv --python 3.12
uv sync
uv run python run_api.py
```

```bash
cd app/frontend
bun install
bun dev
```
