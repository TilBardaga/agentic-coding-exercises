# Agent jobs

Learners and facilitators use this folder for **`/plan`** output: one directory per job, typically starting with **`dev-plan.md`**.

This **published** repo ships the folder **empty** so no maintainer job history is included. Create **`docs/agent-jobs/<job-name>/`** when you run **`/plan`**. See **`.cursor/commands/plan.md`** and **`.cursor/commands/implement.md`** for the full layout (optional **`execution-report.md`**, **`code-review.md`**, **`system-review.md`** in the same job directory).
